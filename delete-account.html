계정 및 데이터 삭제 안내

최종 수정일: 2026년 4월 18일  |  발효일: 2026년 4월 18일

FlamePunch Studio (이하 "회사")가 개발 및 운영하는 모바일 게임 스톤 타워 디펜스: 영웅의 증명 (이하 "Stone TD" 또는 "서비스")의 이용자는 언제든지 자신의 계정과 관련 데이터의 삭제를 요청할 수 있습니다. 본 페이지는 계정 삭제 절차, 삭제되는 데이터의 유형, 보관되는 데이터의 종류 및 보관 기간을 안내합니다.

1. 인앱에서 직접 삭제 (권장)

가장 빠르고 간단한 방법입니다. 게임 내에서 다음 절차를 따라주세요.

  • Stone TD 앱 실행
  • 메인 메뉴 → 우측 상단 ⚙️ 설정 버튼 터치
  • 설정 메뉴 내 "계정 삭제" 버튼 터치
  • 경고 팝업에서 "삭제 확인" 선택
  • 처리 완료 후 게임이 자동으로 초기화되며, 새 계정으로 다시 시작할 수 있습니다

※ 인앱 삭제 요청은 즉시 처리되며, 클라우드 백업 데이터는 30일 이내에 완전히 파기됩니다.

2. 이메일 요청을 통한 삭제

앱을 더 이상 설치하지 않은 경우, 또는 인앱 삭제가 어려운 경우 다음 이메일로 삭제를 요청해 주세요.

  • 수신 이메일: support@flamepunch.com
  • 제목: [계정 삭제 요청]
  • 본문에 포함할 정보:
    • 등록된 게임 내 닉네임
    • Google 로그인 사용 시: Google 계정 이메일 주소
    • (선택) PlayFab Player ID — 알고 있는 경우
    • (선택) 마지막 접속 대략적 날짜 — 본인 확인 보조용

※ 회사는 본인 확인 후 7일 이내에 1차 답변을 드리며, 30일 이내에 모든 데이터의 삭제를 완료합니다.

3. 삭제되는 데이터

계정 삭제 요청 시 다음 데이터가 모두 삭제됩니다.

데이터 유형 저장 위치 처리
PlayFab 계정 (PlayerID, 닉네임, 로그인 이력) Microsoft Azure (PlayFab) 완전 삭제
클라우드 세이브 데이터 (영웅 보유, 재화, 진행 상황) Microsoft Azure (PlayFab) 완전 삭제
Google 계정 연동 정보 (sub ID, 이메일, 표시 이름) Microsoft Azure (PlayFab) 완전 삭제
Firebase 사용자 ID 및 사용자 속성 Google Firebase 완전 삭제
광고 ID 연결 정보 (AdMob) Google AdMob 연결 해제
리더보드 등록 점수 (닉네임 기반) Microsoft Azure (PlayFab) 완전 삭제
로컬 기기 저장 데이터 (앱 캐시, 토큰) 이용자 기기 앱 삭제 시 자동 제거

4. 보관되는 데이터 및 보관 기간

관련 법령 또는 부정 이용 방지 목적으로 다음 데이터는 일정 기간 보관 후 파기됩니다. 모든 보관 데이터는 개인을 식별할 수 없도록 익명화되거나 별도의 안전한 저장소에 격리됩니다.

데이터 유형 보관 기간 법적 근거
결제 기록 (구매 내역, 영수증) 5년 전자상거래 등에서의 소비자보호에 관한 법률 제6조
부정 이용 기록 (사기, 어뷰징 등) 1년 재가입 차단 및 회사의 정당한 이익 보호
익명화된 통계 데이터 (Firebase Analytics) 최대 14개월 Google Firebase 데이터 보존 정책

5. 자주 묻는 질문

Q1. 삭제 후 데이터를 복구할 수 있나요?

삭제 요청 후 30일이 경과하면 모든 데이터는 영구적으로 소실되며 복구가 불가능합니다. 30일 이내에는 동일 Google 계정으로 재로그인 시 부분적인 복구가 가능할 수 있으나, 보장되지 않습니다.

Q2. 결제한 젬은 환불되나요?

이미 사용한 인앱 재화(젬, 별가루 등)는 계정 삭제와 무관하게 환불되지 않습니다. 결제 자체에 대한 환불은 결제 수단 제공자의 환불 정책을 따릅니다.

  • Google Play 환불: Google Play 환불 정책
  • 원스토어 환불: 원스토어 환불 안내

Q3. 새 계정을 만들 수 있나요?

네, 계정 삭제 후에도 동일 기기 또는 동일 Google 계정으로 새로운 계정을 만들어 게임을 다시 시작할 수 있습니다. 단, 이전 계정의 진행 상황(영웅, 재화, 업적 등)은 새 계정으로 이전되지 않습니다.

Q4. 일부 데이터만 선택적으로 삭제할 수 있나요?

현재 Stone TD 는 전체 계정 삭제만 지원합니다. 광고 ID 재설정 등 일부 데이터에 대한 부분 제어는 OS 레벨 설정에서 가능합니다.

  • Android 광고 ID 재설정: 설정 → Google → 광고 → 광고 ID 재설정
  • Google 계정 권한 회수: Google 계정 설정 에서 Stone TD 의 접근 권한 철회

Account & Data Deletion

Last updated: April 18, 2026  |  Effective: April 18, 2026

FlamePunch Studio ("we", "us", or "our"), the developer and operator of the mobile game Stone TD: Proof of Heroes (the "Service"), allows users to request deletion of their account and associated data at any time. This page describes the account deletion procedure, the data types that will be deleted, and the data types that will be retained along with retention periods.

1. In-App Deletion (Recommended)

The fastest and simplest method. Follow these steps within the game:

  • Launch the Stone TD app
  • From the main menu, tap the ⚙️ Settings button at the top right
  • In the Settings menu, tap the "Delete Account" button
  • On the warning popup, tap "Confirm Delete"
  • The game will automatically reset upon completion. You may start fresh with a new account at any time

※ In-app deletion requests are processed immediately, and cloud backup data is fully purged within 30 days.

2. Email Request

If you have already uninstalled the app or in-app deletion is not feasible, please send a deletion request via email:

  • Recipient: support@flamepunch.com
  • Subject: [Account Deletion Request]
  • Required information in the body:
    • Your in-game nickname
    • If using Google Sign-In: your Google account email
    • (Optional) PlayFab Player ID — if known
    • (Optional) Approximate date of last login — for identity verification

※ We will provide an initial response within 7 days and complete deletion of all data within 30 days after identity verification.

3. Data That Will Be Deleted

Upon account deletion, the following data will be fully removed:

Data Type Storage Location Action
PlayFab account (PlayerID, nickname, login history) Microsoft Azure (PlayFab) Fully deleted
Cloud save data (heroes owned, currency, game progress) Microsoft Azure (PlayFab) Fully deleted
Google account linkage (sub ID, email, display name) Microsoft Azure (PlayFab) Fully deleted
Firebase user ID and user properties Google Firebase Fully deleted
Advertising ID linkage (AdMob) Google AdMob Linkage severed
Leaderboard entries (nickname-based) Microsoft Azure (PlayFab) Fully deleted
Local device storage (app cache, tokens) User device Removed automatically when app is uninstalled

4. Data That Will Be Retained

For legal compliance and fraud prevention, the following data will be retained for a defined period before being purged. All retained data is anonymized or isolated in secure storage so that individuals cannot be identified.

Data Type Retention Period Legal Basis
Payment records (purchase history, receipts) 5 years Korean Consumer Protection Act (Article 6 of the Act on Consumer Protection in Electronic Commerce)
Fraud / abuse records 1 year Re-registration prevention and legitimate interests of the operator
Anonymized analytics data (Firebase Analytics) Up to 14 months Google Firebase data retention policy

5. Frequently Asked Questions

Q1. Can I recover my data after deletion?

After 30 days from the deletion request, all data is permanently lost and cannot be recovered. Within the first 30 days, partial recovery may be possible by signing in with the same Google account, but it is not guaranteed.

Q2. Will my purchased gems be refunded?

In-app currencies (gems, stardust, etc.) that have already been used are not refundable, regardless of account deletion. Refunds for the purchases themselves are subject to the refund policies of the respective payment providers:

  • Google Play refunds: Google Play Refund Policy
  • ONE Store refunds: ONE Store Refund Guide

Q3. Can I create a new account?

Yes. After deletion, you may create a new account on the same device or with the same Google account and start fresh. However, the previous account's progress (heroes, currency, achievements, etc.) will not be transferred to the new account.

Q4. Can I selectively delete only some of my data?

Stone TD currently only supports full account deletion. Partial control over certain data, such as advertising ID, is available at the OS level:

  • Reset Android Advertising ID: Settings → Google → Ads → Reset advertising ID
  • Revoke Google account permissions: visit Google Account Permissions and revoke Stone TD's access
